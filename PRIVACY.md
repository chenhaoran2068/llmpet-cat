# 隐私说明

## 猫猫小窝数据

猫窝、每日纪念册、专注契约与休息打卡只保存在本机应用的浏览器本地存储中。工作小道具只参考 Codex 的工具名称与本次工具调用次数，不会把任务正文写入猫窝或周报。

周日自动生成或手动生成的“本周明信片”只会保存为本机 `图片\\LLMPET Cat` 文件夹中的 PNG 图片；不会上传网络，也不包含 Token、价格或预算统计。

LLMPET Cat 仅在本机读取 Codex 保存的会话文件：
`~/.codex/sessions/**/*.jsonl`。读取内容只用于判断桌宠状态和显示本地会话
名称；程序不会把会话内容上传到网络，也不包含网络价格同步或遥测功能。

程序本地设置和日志保存在 `~/.octopus/`，包括猫猫位置、开机自启偏好和
运行日志。该目录是为兼容早期 0.2.x 版本保留的历史名称。

为显示“陪你工作中”和休息提醒，程序会读取 Windows 提供的**系统空闲时长**。
它不会记录按键内容、屏幕内容、正在使用的软件名称或浏览历史。待机陪伴时会临时
读取鼠标位于猫猫左侧或右侧的方向，让猫猫偶尔看过去；不会保存鼠标坐标。

“猫猫啃屏幕”是本地视觉覆盖效果。覆盖层设置为忽略鼠标事件，不会截获
键盘、鼠标输入，不会删除文件、关闭窗口或修改屏幕设置。
