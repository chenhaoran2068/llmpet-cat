# 隐私说明

LLMPET Cat 仅在本机读取 Codex 保存的会话文件
`~/.codex/sessions/**/*.jsonl`，以及 Kimi Desktop 的本地会话记录。读取内容只用于判断项目状态、显示本地项目名称和选择猫猫动作；程序不会上传会话内容，也不包含网络价格同步或遥测功能。

程序本地设置和日志保存在 `~/.octopus/`；项目猫的显示、名字和身份偏好保存在应用本地存储中。它们只保存在本机。

为显示“陪你工作中”和休息提醒，程序会读取 Windows 提供的**系统空闲时长**。
它不会记录按键内容、屏幕内容、正在使用的软件名称或浏览历史。待机陪伴时会临时
读取鼠标位于猫猫左侧或右侧的方向，让猫猫偶尔看过去；不会保存鼠标坐标。

“猫猫啃屏幕”是本地视觉覆盖效果。覆盖层设置为忽略鼠标事件，不会截获
键盘、鼠标输入，不会删除文件、关闭窗口或修改屏幕设置。
